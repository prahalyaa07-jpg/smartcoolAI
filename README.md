# SmartCoolAI

SmartCoolAI is an AI-powered Urban Heat Mitigation Dashboard.

## Features

- AI Heat Risk Prediction
- Interactive Heat Hotspot Maps
- National Heat Risk Leaderboard
- Cooling Intervention Simulator
- Multi-City Analysis (Chennai, Bengaluru, Hyderabad)

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Folium
- Scikit-Learn

## Objective

Help city planners identify urban heat hotspots and recommend cooling strategies using AI and geospatial analytics.
